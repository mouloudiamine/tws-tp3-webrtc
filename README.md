### Usage
#### Preparation
1. installer les packages npm 
```bash
npm install
```
2. build 
```bash
npm build 
```
#### Production
Demarer le serveur
```bash
npm start 
```
#### Developpement
 Lancer en mode dev
```bash
npm run dev 
```
#### Deploiement
Il se fait d'associer le repo github de l'application avec Heroku
et à chaque push le déploimement se fait automatiquement

#### Test
Environnement de test est le logiciel chrome
